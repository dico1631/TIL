# 정규표현식

> 모든 언어에 공통적으로 적용되는 특수 언어로, 비밀번호/이메일 등 정해진 규칙에 맞는 내용을 입력했는지를 검사할 때 사용한다.



### 1. 기본 사용 방법

```python
import re
pattern = re.compile('검사할 문자열')
print(pattern.match('정규표현식'))
```



### 2. 정규표현식



#### 1) [ ] 

- or의 의미 : [com] > c,o,m 중에 하나라도 나오면 T 아니면 None이 나옴

- 두 문자 사이에 하이픈(-)을 사용하면 범위(From - To)를 의미

  : [a-z] > 소문자 전체, [A-Z] > 대문자 전체, [0-9] > 숫자 전체

- [^a-z] : 소문자 외의 것, [ ] 안에 ^는 not의 의미



#### 2) dot( . )

- 줄바꿈 문자 \n을 제외한 모든 문자가 1개 들어감을 의미
- [.]  : [ ] 안에 dot( . )이 들어가면 문자 그대로 dot( . )을 의미



#### 3) * / +

- 별표(*) : 바로 앞 문자가 0~n번 등장함을 의미 (없어도 됨)
- 플러스(+) : 바로 앞 문자가 1~n번 등장함을 의미 (무조건 1회 이상)



#### 4) {m} / {m,n}

- {m} : 앞에 나온 문자가 m번 나옴
- {m,n} : 앞의 문자가 m~n번 나옴



#### 5) ([ ] 없이) ^ / $

- ^ : 뒤쪽에 나오는 패턴으로 무조건 시작해야함을 의미

- $ : 앞에 나오는 패턴으로 무조건 끝나야 함을 의미

  

#### 6) or ( | )

- 여러 패턴 중에 하나



### 3. 정규식 그룹화

>  ( ) 로 원하는 부분만 그룹화 시킬 수 있다. 그룹화를 하면 index로 접근이 가능하다.

- ( )로 원하는 부분을 묶으면 그룹화가 됨
- group(index) 함수로 선택 / group() = group(0) = 그룹 상관없이 전체 문자열



### 4. 관련 함수

#### 1) match

- **첫 글자**부터 정규식과 매치되는 지 확인 > 시작 부분이 정규식과 일치해야 함 (**1개**)



#### 2) search

- **전체에서** 정규식과 매치되는 부분이 있는지 확인 > 문자열 중 정규식과 일치되는 부분이 있기만 하면 됨 (**1개**)



### 3) findall

- 정규식과 일피되는 **모든** 문자열을 list로 반환 (**모두**)
- group을 하면 전체 문자열에서 정규식과 일치하는지 확인하고 반환할 때는 group한 부분만 반환
