 <자바 함수 사용>
- static 
 static이 붙어 있는 메소드는 같은 클래스의 다른 메소드를 부를 수 없다.
static이 붙어 있으면 메모리에 제일 먼저 등록하기 때문에 다른 메소드나 변수 등의 존재를 알지 못한다.
new를 해야만 메모리에 등록되는 거기에 new를 해서 메모리에 등록시킨 다음에 사용하거나,
static을 붙여서 사용 될 것도 메모리에 먼저 등록시킨다.
