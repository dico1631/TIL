<이클립스 사용법>
1. project explorer 지웠을 때 :
window > perspective > reset perspective

2. workspace 위치 알기 :
File > switch workspace

3. perspective : 내가 보는 장면

4. syso + control : System.out.println();

5. main + control : main 메소드

6. F11 + control : 실행

7. shift + control + F : 자동정렬

8. shift + control + C  or  control + / : 주석, 여러 줄 한번에 주석처리 가능
